# PC Builder Mockup

React + Tailwind SPA that lets you explore the home screen, start a build, view saved builds, and access auth/support pages through the navbar. Routing is handled by React Router, and global styles/state live under `src/`.

## Requirements
- Node.js 18+ (ships with npm)
- Modern browser for `http://localhost:3000`

## Run locally
```bash
npm install
npm start
```
